# NeoVim config

### Using colorscheme
*Monokai Pro - octagon*

### Mappings
`<leader>` - `SPACE`
#### Telescope
- `<leader>ff` - find file
- `<leader>fg` - Live Grep
#### Comments
- `<leader>c` - line comment
- `<leader>/` - block comment

#### Navigation
- `<leader>gd` - goto definition
- `<leader>tf` - neotree float focus

#### Basic
- `<leader>|` - vertical split
